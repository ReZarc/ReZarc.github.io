D - Link with Game Glitch

二分 + spfa 判负环。将所有的 $B → D$ 花费的 $a$ 个 $b$ 和 $c$ 个 $d$ 折合成 $B = c / a * D$ 然后将其求个 $log$ 进行转化，这样求 $w$ 时就能直接加而不是乘，二分找最小的没有负环的 $w$ 。

G - Link with Monotonic Subsequence

找规律的签到，发现 $sqrt(n)$ 分块输出即可。

J - Link with Arithmetic Progression

公式题。给你一个序列，求将其变成等差序列的最小 $∑(a[i] - a’[i])^2$

简化一下，给你一堆散点，画出线性回归曲线…… 有那味了，但是不会写，公式早忘了。

后来在乱翻书的时候发现高数下的最小二乘法，抄了方程解一下即可。

看别人假了这么多发还以为卡常卡精度，加了一堆没有用的东西。

听说出题人刚开始是写了标程再出的题，本意是三分套三分（。

K - Link with Bracket Sequence I

dp。$f[i][j][k]$ 代表 $b[i]$ 与 $a[j]$ 相同位置的匹配，还有 $k$ 个 ‘$($’ 未匹配。

L - Link with Level Editor I

dp。赛时往图论上想了，赛后发现人家都告诉你了内存只给32M，存不下 1e4 * 2e3 的图，意图是让往滚动数组dp之类的方法上想。

而且赛时还把题读假了，天天上演莫比乌斯反演队友。

不考虑内存的话开个 $f[N][M]$。

$f[i][j]$ 表示考虑到第 $i$ 个世界，目前在 $j$ 的最短区间长度。

将转移分为走给出的边和直接不动两种转移即可，由于空间的限制需要压掉第一维。
